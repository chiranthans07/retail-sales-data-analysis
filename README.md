# Retail Sales Data Analysis

## Overview
This project performs Exploratory Data Analysis (EDA) on a retail sales dataset to understand customer purchasing behavior and identify sales trends.

## Objectives
The main goal of this project is to analyze retail sales data and extract meaningful insights using Python.

## Tools and Libraries
- Python
- Pandas
- Matplotlib
- Seaborn

## Dataset
The dataset contains information related to retail transactions including:
- Customer Age
- Gender
- Product Category
- Purchase Amount

## Analysis Performed
The following steps were performed during the analysis:

1. Data loading and exploration
2. Dataset structure analysis
3. Statistical summary of numerical features
4. Visualization of product category sales
5. Gender-based purchase analysis
6. Age vs purchase behavior analysis

## Visualizations
The project includes multiple visualizations such as:
- Product category sales distribution
- Purchase amount distribution
- Gender-based purchase analysis
- Age vs purchase amount scatter plot

## Key Insights
- Some product categories show higher sales frequency.
- Purchase behavior varies across customer demographics.
- Sales distribution reveals patterns in transaction values.

## Conclusion
Exploratory Data Analysis helps businesses understand customer behavior and make data-driven decisions.
